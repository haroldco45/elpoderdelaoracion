# 🕊️ El Poder de la Mujer que Ora - PWA

Una aplicación web progresiva (PWA) completa, elegante y funcional basada en los principios del libro **"El Poder de la Mujer que Ora"** de Stormie Omartian[cite: 1]. Esta herramienta está diseñada para estructurar un viaje de intercesión y crecimiento espiritual de **30 días**, permitiendo realizar un seguimiento del progreso diario y compartir motivos de oración con precisión geográfica[cite: 1].

Desarrollado bajo los lineamientos metodológicos de estudio profundo y con todos los derechos de autor reservados a nombre de **Vibras Positivas**[cite: 1].

---

## ✨ Características Principales

* 📅 **Guía Estructurada de 30 Días**: Cubre los aspectos fundamentales del libro (el caminar personal, el matrimonio, los hijos, el control de los pensamientos, los temores, la salud y el entorno)[cite: 1].
* 📚 **Enfoque de Estudio de 3 Pilares**: Cada día devocional incluye de forma estricta:
    1.  **Capítulo y Versículo**: La base bíblica que fundamenta el clamor del día[cite: 1].
    2.  **Contexto Histórico / Espiritual**: El trasfondo de las enseñanzas de Stormie Omartian[cite: 1].
    3.  **Aplicación a la Realidad Actual**: Ejercicios prácticos y reflexiones para el día a día contemporáneo[cite: 1].
* 📍 **Geolocalización Obligatoria**: Integración nativa que captura las coordenadas en tiempo real del usuario para georreferenciar el origen de la intercesión y del clamor[cite: 1].
* 💾 **Persistencia Local (LocalStorage)**: Sistema automático que almacena los días completados sin necesidad de bases de datos externas ni registros complejos[cite: 1].
* 📲 **Interacción con WhatsApp**: Botones para compartir el devocional del día o lanzar peticiones de oración de emergencia directamente a grupos o contactos con un solo toque[cite: 1].
* 📱 **Experiencia PWA Nativa**: Diseño adaptativo móvil (Mobile-First) con soporte para instalación en pantalla de inicio y funcionalidad básica fuera de línea (offline)[cite: 1].

---

## 🛠️ Tecnologías Utilizadas

* **HTML5 & CSS3**: Estructura limpia y semántica[cite: 1].
* **Tailwind CSS**: Estilizado moderno, fluido y responsivo con paleta de colores violeta y lavanda que reflejan espiritualidad y serenidad[cite: 1].
* **JavaScript (Vanilla ES6)**: Lógica de la aplicación, manejo de estados, persistencia y consumos de API del navegador[cite: 1].
* **Geolocalización API**: Captura precisa del punto geográfico de oración[cite: 1].
* **Web App Manifest**: Configuración para la instalación directa en dispositivos iOS, Android y Escritorio[cite: 1].

## 🚀 Instalación y Despliegue Rápido

Al ser una aplicación de un solo archivo plano (`index.html`), su despliegue es inmediato[cite: 1]:

1.  **Descarga o copia** el archivo `index.html` provisto en el proyecto[cite: 1].
2.  **Súbelo a tu servidor de preferencia** o utiliza plataformas de despliegue estático gratuitas como[cite: 1]:
    * [Vercel](https://vercel.com)[cite: 1]
    * [Netlify](https://www.netlify.com)[cite: 1]
    * [GitHub Pages](https://pages.github.com)[cite: 1]
3.  **Instala en tu móvil**: Abre el enlace de tu despliegue desde el navegador de tu celular (Safari en iOS o Chrome en Android), pulsa el botón "Compartir" o "Ajustes" y selecciona **"Agregar a la pantalla de inicio"**[cite: 1].

## 📋 Estructura de Datos de los Devocionales

Cada módulo de datos dentro del código sigue de manera rigurosa la siguiente arquitectura semántica[cite: 1]:

```javascript
{
    dia: Number,
    titulo: String,
    versiculo: String,
    texto_versiculo: String,
    contexto: String,      // Contexto Histórico/Espiritual
    aplicacion: String,    // Aplicación a la Realidad Actual
    oracion: String        // Motivo de oración guiado
}
```[cite: 1]
